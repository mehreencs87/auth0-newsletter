Newsletter from Auth0: 2014-07-04
Hi there,

Lots of stuff happened in the past month, and we wanted to share the latest Auth0 updates with you:

WordPress + Auth0

We've just released a Wordpress plugin that will let you do Single Sign On with all of our Enterprise Providers (AD, LDap, etc), Social Providers (Facebook, Twitter, Google, etc.) or with username and password. This plugin will also give you access to Multifactor Auth, our User management dashboard and reporting and analytics. You can read our blog post with the announcement and try out a demo here

AngularJS SDK 1.0

Our AngularJS SDK has reached version 1.0. We've improved the redirect support, we've added requiresLogin parameter to routes so that if the user wants to get to an authenticated route he'll be redirected to login, we've improved our code quality and many many more features. Try it out by installing the newest version from npm or bower.

Auth0 + AngularJS/jQuery tutorial on CodeCademy

If you want to get started with Auth0 using either AngularJS or jQuery, you can do our new CodeCademy course to learn in 8 simple steps how to implement authentication and call a secured API that you make. Follow this link to start the angular tutorial or this other one for the jQuery one

Backendless apps with Auth0 + Firebase

Learn how to create Backendless apps with Authentication using Firebase + Auth0. Read all about it in the guest post we've written on firebase's blog

Auth0 + Appery.io Webinar

Join us on August 6 at 11AM PDF to learn about using Auth0 with Appery to build secure mobile apps in the cloud. Click here to join the Webinar, and check out the blog post that Appery has written about the integration.
